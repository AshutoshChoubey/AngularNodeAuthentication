Please follow the following step to complete the JWT authentication in node js
 

Step 1: Install nodemon

 
nodemon is a tool that helps develop node.js based applications by automatically restarting the node application when file changes in the directory are detected.

nodemon does not require any additional changes to your code or method of development.
 
npm install -g nodemon

Step 2: Start Node js project

 
Usually, when we start a new Node.js project using npm to generate my initial project.

npm init
npm will ask some questions and builds a package.json file. Then we can start building the project.

Step 3:Install Express Js

Please try to type the command so that you will get habituated to typing from mind than searching syntax from the documentation.
Here are the below screenshot instruction where you can follow But make sure read carefully some of the commands you already did so this screenshot for your understanding

npm install express --save

Step 4: Install the Mongoose library:

Mongoose provides a straight-forward, schema-based solution to model your application data. It includes built-in type casting, validation, query building, business logic hooks and more, out of the box.

Next install Mongoose from the command line using npm:

npm install mongoose

Step 5: Install the body-parser :

Node.js body parsing middleware.

Parse incoming request bodies in a middleware before your handlers, available under the req.body property.

npm i body-parser
